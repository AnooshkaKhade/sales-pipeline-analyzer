# Sales Pipeline Analyzer

A python tool that analyzes CRM sales pipeline data.

Features:
pipeline stage breakdown 
Win rate calculation
Revenue analysis

Tech stack:
Python
Pandas

Example Output:

Total records: 10
Win rate: 66.67%
Revenue: 100000
